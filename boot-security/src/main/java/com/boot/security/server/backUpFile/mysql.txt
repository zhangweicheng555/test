日期总结：数据库存储的是日期   不是字符串

mysql字符串转日期  适应于所有日期格式字符串  
统一转为 带秒的日期

mysql> select str_to_date('2018-09-09 14:12:56','%Y-%m-%d %H:%i:%s') from dual;
+--------------------------------------------------------+
| str_to_date('2018-09-09 14:12:56','%Y-%m-%d %H:%i:%s') |
+--------------------------------------------------------+
| 2018-09-09 14:12:56                                    |
+--------------------------------------------------------+
1 row in set (0.00 sec)

mysql> select str_to_date('2018-09-09 14:12','%Y-%m-%d %H:%i:%s') from dual;
+-----------------------------------------------------+
| str_to_date('2018-09-09 14:12','%Y-%m-%d %H:%i:%s') |
+-----------------------------------------------------+
| 2018-09-09 14:12:00                                 |
+-----------------------------------------------------+
1 row in set (0.00 sec)

mysql> select str_to_date('2018-09-09 14','%Y-%m-%d %H:%i:%s') from dual;
+--------------------------------------------------+
| str_to_date('2018-09-09 14','%Y-%m-%d %H:%i:%s') |
+--------------------------------------------------+
| 2018-09-09 14:00:00                              |
+--------------------------------------------------+
1 row in set (0.00 sec)

mysql> select str_to_date('2018-09-09 ','%Y-%m-%d %H:%i:%s') from dual;
+------------------------------------------------+
| str_to_date('2018-09-09 ','%Y-%m-%d %H:%i:%s') |
+------------------------------------------------+
| 2018-09-09 00:00:00                            |
+------------------------------------------------+
1 row in set (0.00 sec)


--将日期转为你自己需要的那种格式
date_format(date, format) 函数，MySQL日期格式化函数date_format()
date_format(str_to_date('2018-09-09 14','%Y-%m-%d %H:%i:%s'), '%Y%m%d %H%i%s')
